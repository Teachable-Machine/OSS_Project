# 탈모 측정기

## About The Project
  탈모에 대해 걱정이 많은 사람들을 위해 탈모의 진행정도를 보여주는 웹 페이지
> * 사진을 업로드하여 탈모를 판별
> * 딥러닝 모델을 이용해 이미지로 탈모 단계 분류

## Getting Started
1. node.js install
2. git clone https://github.com/Teachable-Machine/OSS_Project
3. Install dependencies
```
npm install express
npm install express-fileupload
```
4. `node server.js`
